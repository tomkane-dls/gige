# gige

GigE camera acquisition script. Configures the camera via EPICS PVs and saves frames as TIFF files.

## Usage

```
uv run gige.py config.yaml
uv run gige.py --help
uv run gige.py -v config.yaml   # verbose/debug logging
```

## Configuration

All parameters are set in a YAML file. See `config.yaml` for an example with descriptions of each field.
